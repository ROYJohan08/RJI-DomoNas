## Montage.
- Branchez l'onduleur sur une prise 230v.
- Branchez l'ordinateur a l'onduleur.
- Branchez l'alimentation a l'onduleur.
- A l'aide d'un cable en cuivre fin, reliez le fil vert de l'alimentation ATX secondaire a u fil vert de l'alimentation principale.
  - Ce fil permet de démarrer les deux alimentation au meme moment afin de gérer l'allumage des disques dur.
- Branchez tous les disques dur a l'alimentation secondaire.
- Branchez les sata sur la carte additionnelle PCIE.
- Branchez la carte PCIE a l'ordinateur.
- Branchez les ventilateurs a l'USB et placez les sous les disques dur.
